## Setup

This site uses Jekyll with the midnight theme. To run locally:

1. Install Ruby and Bundler
2. Run `bundle install` to install dependencies
3. Run `jekyll serve` to start the development server
4. Visit `http://localhost:4000`

For deployment to GitHub Pages, the site is configured to use the `jekyll-theme-midnight` gem.
